How it works?

D3.js is used for data visiualization which is a JavaScript library for producing dynamic, interactive data visualizations in web browsers. It makes use of Scalable Vector Graphics, HTML5, and Cascading Style Sheets standards. The data is being stored in Firestore.

  • How to add a data? : Fill inputs and hit enter or click "Add"
  • How to delete a data? : Hover over the data you want to delete on graph and click and type "delete"
